# DataEval Workflows

Data evaluation and quality inspection tools, built on [DataEval](https://github.com/aria-ml/dataeval/).

DataEval Workflows wraps the DataEval library into a configurable, containerized application
for dataset loading, quality analysis, and data cleaning workflows.

## Key Features

- **Dataset Loading** — Load and inspect datasets in MAITE-compatible format
- **Data Quality Analysis** — Detect outliers, duplicates, and label issues
- **Configurable Workflows** — YAML-driven data cleaning pipelines
- **Container Deployment** — GPU and CPU Docker images for production use
